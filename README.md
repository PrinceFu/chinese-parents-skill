## 这是什么？

一套基于学术研究、纪录片、文化观察深度蒸馏的**中国式父母分析框架**。不是定义"中国父母是什么样的"，而是提供一套思维透镜——

| 使用场景 | 怎么用 |
|---------|--------|
| **写小说/剧本** | 用心智模型构建角色动机，用话语DNA写对话，用决策启发式设计冲突场景 |
| **心理学/教育学研究** | 用学术概念（行为控制vs心理控制、互惠性孝道vs权威性孝道）分析案例 |
| **理解自己的父母** | 用心智模型理解父母行为背后的逻辑，用内在张力解释"又爱又恨"的矛盾感 |
| **与中国式父母对话** | 用话语DNA预判父母的典型反应，用决策启发式理解他们的价值排序 |
| **社会学/文化分析** | 用历史演变理解"不变的深层结构"，用跨文化对比理解独特性 |

## 框架概览

### 5个心智模型

| 模型 | 一句话 | 适用场景 |
|------|--------|---------|
| 教育=阶层跃迁的唯一通道 | 从科举到高考到学区房，"读书"是跨越千年的阶层流动信仰 | 分析教育焦虑、学区房、补习班 |
| 子女=家庭的核心投资品 | 子女不是独立个体，而是家庭资产的增值部分 | 理解职业/婚恋/生育干预 |
| 自我牺牲=道德权威 | 父母的付出构成不可偿还的"道德债务" | 解读"我都是为你好" |
| 集体焦虑驱动>个体理性 | "别人怎么做"是决策的核心参照 | 分析从众行为、剧场效应 |
| 血缘=无边界 | 亲子关系中不存在隐私和独立人格 | 理解边界侵犯、妈宝现象 |

### 8条决策启发式

经济杠杆控制 · 比较式驱动 · 未来灾难化 · 情感撤回威胁 · 延迟满足的空头支票 · 经验权威压制 · 反问终结讨论 · 从众安全策略

### 7个话语DNA

权威预设 · 苦难叙事 · 比较焦虑 · 结果导向 · 否定前置 · 边界模糊 · 未来威胁

### 特殊表达系统

**食物代替"我爱你"**："吃饭了吗？"= 我想你了 · "多吃点，太瘦了"= 我心疼你 · "我不饿，你吃吧"= 我愿意为你牺牲

**"出来吃饭了"= 中国式假性道歉**：中国父母极少说"对不起"，用行为代替语言。

## 诚实边界

本框架**能做到**的：
- 提供分析中国式父母行为的思维工具
- 帮助理解父母行为背后的深层逻辑
- 为写作、研究、对话提供框架和素材

本框架**做不到**的：
- 不能替代心理咨询——如有严重心理创伤，请寻求专业帮助
- 不能预测个体差异——中国式父母是文化模式，每个具体家庭有巨大差异
- 不能判断"对错"——框架呈现模式和张力，不做价值裁判
- 不能替代真实对话——理解框架不等于理解你的父母
