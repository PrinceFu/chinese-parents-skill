## これは何？

学術研究、ドキュメンタリー、文化的観察から深く蒸留された**中国式親分析フレームワーク**です。「中国の親とは何か」を定義するのではなく、パターンを理解するための分析ツールセットです。

| 用途 | 使い方 |
|------|--------|
| **小説/脚本執筆** | メンタルモデルでキャラクターの動機を構築し、表現DNAで会話を書き、意思決定ヒューリスティックで対立を設計 |
| **心理学/教育学研究** | 学術的概念（行動的制御 vs 心理的制御、互恵的孝道 vs 権威的孝道）を適用 |
| **自分の親を理解する** | メンタルモデルで親の行動の論理を解読し、内的緊張で「愛憎相半ばする」矛盾を説明 |
| **中国式親との対話** | 表現DNAで典型的な反応を予測し、意思決定ヒューリスティックで価値観の優先順位を理解 |
| **社会学/文化分析** | 歴史的変遷で「変わらない深層構造」を理解し、異文化比較で独自性を把握 |

## フレームワーク概要

### 5つのメンタルモデル

| モデル | 一行説明 | 適用場面 |
|--------|---------|---------|
| 教育 = 階層上昇の唯一の確実な道 | 科挙から大学入学試験まで、「勉強」は千年を超える階層流動への信仰 | 教育不安、学区房、塾の分析 |
| 子供 = 家族の核となる投資品 | 子供は独立した個人ではなく、家族資産の増値部分 | 職業/婚活/出産介入の理解 |
| 自己犠牲 = 道徳的権威 | 親の犠牲は返済不能の「道徳的負債」を生む | 「全部あなたのため」の解読 |
| 集団不安 > 個人の合理性 | 「他の人がどうしているか」が意思決定の核心参照 | 同調行動、シアター効果の分析 |
| 血縁 = 境界なし | 親子関係にはプライバシーも独立した人格も存在しない | 境界侵犯、マザコン現象の理解 |

### 8つの意思決定ヒューリスティック

経済的レバレッジ制御 · 比較駆動 · 未来の悲観化 · 感情撤回の脅威 · 延遲満足の空手形 · 経験的権威の抑圧 · 反問による議論終結 · 同調安全戦略

### 7つの表現DNAパターン

権威の前提 · 苦難の物語 · 比較不安 · 結果志向 · 否定優先 · 境界の曖昧化 · 未来の脅威

### 特殊表現システム

**「ご飯食べた？」= 「愛してる」**: 「ご飯食べた？」= 会いたい · 「もっと食べなさい、痩せすぎ」= 心配してる · 「私はお腹空いてない、あなたが食べなさい」= あなたのためなら犠牲になれる

**「ご飯よ」= 中国式擬似謝罪**: 中国の親は「ごめんなさい」をほとんど言わない——代わりに行動で表現する。

## 正直な境界

このフレームワークが**できること**:
- 中国式親の行動を分析する思考ツールの提供
- 親の行動の背後にある深い論理の解読
- 執筆、研究、対話のためのフレームワークと素材の提供

このフレームワークが**できないこと**:
- 心理カウンセリングの代替——深刻な心理的トラウマがある場合は専門家に相談を
- 個人差の予測——「中国式親」は文化的パターンであり、家族ごとに大きなばらつきがある
- 「正しい/間違い」の判断——フレームワークはパターンと緊張関係を提示し、道徳的評価は行わない
- 実際の対話の代替——フレームワークを理解することが親を理解することではない
