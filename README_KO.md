# 중국식 부모 · 다차원 분석 프레임워크

[中文](README.md) | [English](README_EN.md) | [日本語](README_JA.md)

> 중국식 부모 행동의 이면에 있는 깊은 논리와, 이 논리들이 다양한 시나리오에서 어떻게 나타나는지 이해하기 위한 사고 렌즈.

---

## 이것은 무엇인가?

학술 연구, 다큐멘터리, 문화 관찰을 기반으로 깊이 있게 증류된 **중국식 부모 분석 프레임워크**입니다. "중국 부모가 무엇인가"를 정의하는 것이 아니라, 패턴을 이해하기 위한 분석 도구 세트입니다.

| 사용场景 | 사용 방법 |
|---------|----------|
| **소설/극본 집필** | 멘탈 모델로 캐릭터 동기를 구축하고, 표현 DNA로 대사를 쓰며, 의사결정 휴리스틱으로 갈등을 설계 |
| **심리학/교육학 연구** | 학술 개념(행동 통제 vs 심리 통제, 호혜적 효도 vs 권위적 효도) 적용 |
| **자신의 부모 이해** | 멘탈 모델로 부모 행동의 논리를 해독하고, 내적 긴장으로 "미워할 수 없는" 모순 설명 |
| **중국식 부모와의 대화** | 표현 DNA로 전형적인 반응을 예측하고, 의사결정 휴리스틱으로 가치 우선순위 이해 |
| **사회학/문화 분석** | 역사적 변천으로 "변하지 않는 심층 구조"를 이해하고, 문화 비교로 독특성 파악 |

## 프레임워크 개요

### 5개 멘탈 모델

| 모델 | 한 줄 설명 | 적용 시나리오 |
|------|----------|-------------|
| 교육 = 계층 상승의 유일한 확실한 통로 | 과거부터 대학 입시까지, "공부"는 천 년을 넘나드는 계층 이동에 대한 신앙 | 교육 불안, 학군 아파트, 과외 분석 |
| 자녀 = 가족의 핵심 투자품 | 자녀는 독립적 개인이 아니라 가족 자산의 증가 부분 | 직업/결혼/출산 개입 이해 |
| 자기 희생 = 도덕적 권위 | 부모의 희생은 상환 불가능한 "도덕적 부채"를 만듦 | "전부 너를 위한 거야" 해독 |
| 집단 불안 > 개인의 합리성 | "남들이 어떻게 하는지"가 의사결정의 핵심 참조 | 동조 행동, 극장 효과 분석 |
| 혈연 = 경계 없음 | 부자 관계에는 프라이버시도 독립적 인격도 존재하지 않음 | 경계 침해, 마마보이 현상 이해 |

### 8개 의사결정 휴리스틱

경제적 레버리지 통제 · 비교 구동 · 미래 비관화 · 감정 철회 위협 · 지연 충족의 공수표 · 경험 권위 억압 · 반문으로 토론 종결 · 동조 안전 전략

### 7개 표현 DNA 패턴

권위 전제 · 고난 서사 · 비교 불안 · 결과 지향 · 부정 우선 · 경계 모호화 · 미래 위협

### 특수 표현 체계

**"밥 먹었어?" = "사랑해"**: "밥 먹었어?" = 보고 싶어 · "더 먹어, 너무 말랐어" = 걱정돼 · "나는 배 안 고파, 네가 먹어" = 너를 위해 희생할 수 있어

**"밥 먹어" = 중국식 가짜 사과**: 중국 부모는 "미안해"를 거의 하지 않는다 — 대신 행동으로 표현한다.

## 솔직한 경계

이 프레임워크가 **할 수 있는 것**:
- 중국식 부모 행동을 분석하는 사고 도구 제공
- 부모 행동 이면의 깊은 논리 해독
- 집필, 연구, 대화를 위한 프레임워크와 소재 제공

이 프레임워크가 **할 수 없는 것**:
- 심리 상담 대체 — 심각한 심리적 트라우마가 있다면 전문가의 도움을 구하라
- 개인차 예측 — "중국식 부모"는 문화적 패턴이며, 가족마다 큰 차이가 있다
- "옳고 그름" 판단 — 프레임워크는 패턴과 긴장을 제시하며, 도덕적 평가를 하지 않는다
- 실제 대화 대체 — 프레임워크를 이해하는 것이 부모를 이해하는 것이 아니다
